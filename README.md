# KOSPI Stock PWA

코스피 국내 주식의 가격 흐름, 기술적 지표, 거래량, 외국인·기관 수급 데이터를 함께 확인할 수 있는 Next.js 기반 PWA 주식 분석 앱입니다.

## 주요 기능

### 1. 종목 분석

- 코스피 종목 코드 입력
- 예시: `005930.KS`, `000660.KS`, `035420.KS`
- 코스피 지수 예시: `^KS11`
- 분석 기간 선택
  - 1개월
  - 3개월
  - 6개월
  - 1년

### 2. 기본 가격 정보

- 현재가
- 전일 대비 %
- 전일 대비 가격
- 분석 신호

### 3. 기술적 분석 지표

현재 앱에서 계산하는 기술 지표는 다음과 같습니다.

- SMA20
- SMA60
- RSI14
- MACD
- Signal
- Histogram
- Bollinger Band
- OBV
- Fear & Greed 자체 참고 점수
- 매물대 차트
- 5일 단순 시뮬레이션

### 4. 한투 API 기반 수급 분석

한국투자증권 Open API를 이용해 외국인·기관 순매수 데이터를 조회합니다.

현재 표시 항목:

- 외국인 최근 5일 순매수
- 기관 최근 5일 순매수
- 외국인 + 기관 최근 5일 합산
- 외국인 + 기관 최근 20일 합산
- 최근 일자별 외국인·기관 수급 표
- 수급 판정
- 기술 + 수급 조합 판정
- 수급 다이버전스 후보 여부

### 5. 관심종목 저장

브라우저 localStorage를 이용해 관심종목을 저장합니다.

- 관심종목 저장
- 관심종목 선택 분석
- 관심종목 삭제

### 6. PWA 지원

`next-pwa`를 사용해 PWA 설치를 지원합니다.

## 데이터 구조

### 가격 데이터

현재 가격 및 차트 데이터는 Yahoo Finance Chart API를 사용합니다.

사용 목적:

- 기간별 종가 데이터
- 거래량 데이터
- 차트 표시
- 기술 지표 계산

### 수급 데이터

외국인·기관 순매수 데이터는 한국투자증권 Open API를 사용합니다.

사용 API:

- 현재가 테스트 API
- 투자자 수급 조회 API
